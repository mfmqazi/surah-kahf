# Surat-ul-Kahf: The Illumination

A premium web application for reciting and learning the first 10 and last 10 verses of Surat-ul-Kahf.

## Features
- **Verse Display**: Clear Arabic text, English transliteration, and translation.
- **Audio Playback**: Listen to verses with a selection of world-renowned reciters.
- **Dual Sections**: Easily switch between the first 10 and last 10 verses.
- **Hadith Insights**: Learn about the benefits of Surat-ul-Kahf.
- **Premium Design**: A modern, glassmorphism-inspired interface with a spiritual ambiance.

## Technologies
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+, Async/Await)
- Quran.com API v4

## How to Run
Simply open `index.html` in your web browser. No build step required.
